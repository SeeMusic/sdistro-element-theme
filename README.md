# sdistro-element-theme

自定义 ElementUI 样式。仅提供 SCSS 文件及相关资源；使用 `storybook` 做控件样式预览

目前版本依赖：
* `ElementUI` 2.4.5+
* `@storybook/vue` 6.4.9

> ElementUI 2.15.6 有一些破坏式的 DOM 结构修改，此处直接做了额外的覆写。

## 本地预览

```
npm run storybook:serve
```

## 打包为静态文件

```
npm run storybook:build
```
